# Grocery Inventory Management System

## 📌 Description
This project is a Java-based Grocery Inventory Management System using JDBC and MySQL.  
It allows shopkeepers to manage products and customers to place orders efficiently.

---

## 🚀 Features

### 🛒 Product Management
- Add new products
- Update stock and expiry date
- Change price and discount
- View low stock products
- Check products nearing expiry

### 👤 Customer System
- Add customer details
- View customer orders

### 📦 Order System
- Place orders
- Multiple product ordering
- Automatic stock update
- Sales tracking
- Total amount calculation

---

## 🛠️ Technologies Used
- Java
- JDBC
- MySQL

---

## ▶️ How to Run

1. Open MySQL Workbench
2. Import `database.sql`
3. Open project in IntelliJ
4. Run `GroceryInventory2.java`

---

## 📊 Database Tables
- product
- customer
- orders
- ordered_items
- suggestion
